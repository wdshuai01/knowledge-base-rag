# -
我们项目目前实现的是一个完整的企业知识库 RAG 问答系统，支持上传 .txt、.md、.pdf 文档，后端会对文档进行保存、解析、清洗、父子 chunk 切分、调用百炼 embedding 向量化并写入 Elasticsearch；用户提问时，系统会同时使用 BM25 关键词检索和向量 KNN 语义检索进行混合召回，再对命中的 child chunk 扩展 parent 上下文和相邻 chunk，拼接 prompt 后调用百炼大模型生成答案，并返回引用来源。系统还实现了文档内容 hash 去重、文档列表查看与删除、重复文档清理、问答耗时日志、历史问答记录、健康检查，以及基于评测集的 Recall@K 检索评测，前端提供了上传、问答、引用展示、文档管理、监控和评测结果展示等完整交互。
